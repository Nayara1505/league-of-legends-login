# league-of-legends-login

📋 **Descrição:** Tela de login do League of Legends recriada com HTML5, CSS3 e JavaScript. O projeto reproduz fielmente o layout original da Riot Games, com formulário de autenticação, opções de login social e wallpaper temático do jogo.


🚀 **Funcionalidades Principais:**

* 🔐 **Formulário de Login** - Campos para nome de usuário e senha com labels flutuantes
* 🌐 **Login Social** - Botões para Facebook, Google, Apple, Xbox e PlayStation
* ☑️ **Checkbox Customizado** - Opção "Manter login" com estilo próprio
* 🖱 **Botão de Submit Dinâmico** - Habilitado/desabilitado conforme preenchimento dos campos
* 🖼 **Wallpaper Temático** - Imagem lateral com arte do jogo
* 📱 **Design Responsivo** - Adaptado para diferentes tamanhos de tela


🛠 **Tecnologias Utilizadas:**

* **Estrutura:** HTML5
* **Estilização:** CSS3
* **Layout:** Flexbox
* **Interatividade:** JavaScript
* **Fontes:** Google Fonts (Lexend Deca)


📱 **Responsividade:**

* ✅ **Desktop** - Exibe formulário e wallpaper lado a lado
* ✅ **Tablet** - Ajuste automático dos elementos para melhor visualização
* ✅ **Mobile** - Reorganização dos campos para melhor usabilidade


🎨 **Recursos de Interface:**

* ✅ Labels com efeito de transição ao focar nos inputs
* ✅ Destaque visual ao focar nos campos (mudança de borda e fundo)
* ✅ Checkbox customizado com estado :checked
* ✅ Botão de submit com estados habilitado e desabilitado
* ✅ Layout moderno utilizando Flexbox
* ✅ Ícones de login social com cores próprias de cada marca
